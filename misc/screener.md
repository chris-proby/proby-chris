# Quota & Screener Survey
**For vendor use · n=12 · Fieldwork: April 20–24, 2026**

---

## Target Audience Overview

| 기준 | 요건 |
|------|------|
| **나이** | 25–39세 (엄격 적용) |
| **성별** | 남성 50% / 여성 50% (±10% 허용) |
| **거주지** | 수도권 (서울특별시, 인천광역시, 경기도) |
| **여행 계획** | 향후 3개월 이내 제주도 또는 후쿠오카 레저 여행 계획 있음 · 숙박 미예약 |
| **여행지 할당** | 제주 7명 / 후쿠오카 5명 |
| **여행 형태** | 솔로 또는 커플 여행 · 제주는 전원 커플, 후쿠오카는 솔로/커플 혼합 |
| **고려 숙박 유형** | 제주 = 펜션 포함 / 후쿠오카 = 아파트·집 전체 포함 |
| **의사결정권자** | 본인이 숙박을 직접 결정하는 주체 |
| **플랫폼 인지** | 에어비엔비 인지 필수 |
| **에어비엔비 예약 이력** | 미예약 8명 / 기예약 4명 |
| **경쟁 플랫폼** | 지난 1년 이내 야놀자·여기어때·네이버·아고다 중 1곳 이상 예약 이력 |
