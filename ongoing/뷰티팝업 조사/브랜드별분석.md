# 뷰티팝업 — 브랜드별 반응 분석

> **분석 기반:** 조연우(세션2), 고명지(세션5), 조연서(세션7) 개별 인터뷰 + 현장 녹음  
> **주의사항:** STT 변환 특성상 브랜드명 혼용·오인 가능성 있음. 일부 항목은 "팝업 내 입점 브랜드"인지 "제품 라인명"인지 원문 기준으로 불확실한 경우 명시.

---

## 팝업 구조 파악

조연우가 입장 직후 언급: **"브랜드가 여러 개가 있는 거구나"**  
→ 이 팝업은 단일 브랜드 팝업이 아닌, **누미(Numi) 그룹 소속 복수 브랜드 동시 입점** 형태로 운영된 것으로 파악됨.

| 브랜드 | 확인 근거 | 제품 카테고리 |
|--------|---------|------------|
| **누미(Numi)** | 팝업 공식 명칭, 3명 공통 인지 | 스킨케어, 마스크, 패드, 굿즈 |
| **아이미미(i'm meme)** | 조연우 사전 인지 언급, 색조 섹션 연관 | 블러셔, 하이라이터, 쉐딩, 색조 전반 |
| **리볼(Revol)** | 조연우 직원 질문 발화 ("리볼 립오일 vs 틴트") | 립오일 전문 라인 |
| **클리오(Clio) 계열** | 고명지 체험 발화 (오일 제형 발색력) | 립 제품 계열 (귀속 불확실) |

## 1. 누미(Numi) — 완도 수산시장 / 김(海苔) 컨셉

### 브랜드 정체성 & 기억도

| 참여자 | 사전 인지 | 핵심 발화 |
|--------|---------|---------|
| 조연우 | 없음 | "완도 돌김 왜 이렇게 웃기냐" → 실구매 |
| 고명지 | 이름만 희미하게 | "이 브랜드가 아니라 그냥 김이 기억이 나" |
| 조연서 | 없음 | "김 컨셉이 너무 좋았어. 김 친절하고 웃기고" |

**→ 브랜드명(Numi)보다 '김' 컨셉이 더 강한 기억 앵커로 작동.**  
3명 전원이 독립적으로 김 컨셉을 최강 인상 포인트 1위로 꼽음.

### 제품 체험 반응

| 제품 | 반응 요약 | 언급자 |
|------|---------|--------|
| 김 패드 (브랜디드 굿즈) | "웃겨서 하나 사봤다, 김" (조연우 실구매) | 조연우, 고명지, 조연서 |
| 클렌징 패드 (양면) | "70매잖아요. 80매 만들어" → 사은품 이벤트로 80매 증정 | 조연서 |
| 노워시 마스크 | "그냥 닦고 주무시면 돼요" → 편의성 긍정 | 조연서 |
| 선크림(톤업) | 아침 데일리로 활용 구상, 조연우·조연서 구매 의향 명시 | 조연우, 조연서 |
| 클렌징 밤 | 조연우 엄마에게 사줄 목적 검토 | 조연우 |

### 스쿱 이벤트 (누미 전용)

- **5만 원 이상 구매 시** 본품을 직접 스쿱으로 퍼 담는 체험형 이벤트
- 조연우 참여 → 구매 금액 상승 트리거
- "직접 참여하는 구조" → 체험 만족도 상승에 기여

### 브랜드 인식 변화

| 참여자 | 방문 전 | 방문 후 |
|--------|---------|---------|
| 조연우 | "나머지도 팝업하니까 알게 된 것 같은데" | "지금 경험은 나쁘지 않은 경험" |
| 고명지 | 이름만 희미하게 인지 | "꾸준하게 다가올 수 있었던 느낌" |
| 조연서 | 전혀 없음 | "판다 그러면 한번 찾아서 사는 거" |

**시사점:** 완도 수산시장/김 컨셉은 브랜드 인지 제로 상태에서도 강력한 첫인상 형성. 단, 브랜드명(Numi)보다 컨셉(김)이 먼저 연상되는 구조 → 2차 구매 연결 위해 브랜드명 각인 보완 필요.

## 2. 아이미미(i'm meme) — 색조 전문 섹션

### 브랜드 인지도 — 유일한 사전 인지 브랜드

> "좋은 아이미미밖에 몰랐는데" — 조연우  
> 고명지·조연서: 사전 인지 없음

- 3명 중 **조연우만 사전 인지** (팝업 방문 전부터 알고 있었음)
- 고명지·조연서는 이번 팝업을 통해 최초 인식

### 제품 체험 반응

| 제품 | 반응 요약 | 언급자 |
|------|---------|--------|
| 블러셔 (크림 제형) | "크림이 있어. 나 요즘 블러셔에 살짝 꽂혀가지고" / "8,900원 가성비" | 조연우 |
| 블러셔 컬러 | "나한테 너무 핑크빛이야" → 개인 컬러 적합성 이슈 | 조연서 |
| 블러셔 형광기 | "약간 형광기가 있다" → 조연우 지적 | 조연우 |
| 스틱 쉐딩 | "턱을 깎고 싶어서" → 3명 공통 관심 | 조연우, 조연서 |
| 하이라이터 (크림) | "크림 제형이 좋아. 대박." (조연서) / "너무 반짝거려. 이건 좀 에바치지" (조연우) — 반응 엇갈림 | 조연우, 조연서 |
| 애교살 라이너 | 탐색 언급 | 조연서 |

### 아이미미 핵심 인사이트

- 사전 인지 브랜드임에도 불구하고 현장 체험 반응은 **혼재** (블러셔 형광기·컬러 적합성 지적)
- 크림 제형에 대한 선호는 3명 공통이나 **개별 컬러 적합성**이 구매 전환 변수
- 아이미미 섹션에서 별도의 특별 이벤트 언급 없음 → 누미 섹션 대비 체험 밀도 낮음

## 3. 리볼(Revol) — 립오일 전문

### 제품 체험

> "리볼 립오일 vs 틴트 차이" — 조연우가 직원에게 직접 질문하여 설명 청취

| 특성 | 체험 반응 |
|------|---------|
| 제형 | 오일 제형 → 얇게 발림, 가벼운 텍스처 |
| 발색 | 5분 경과 후 컬러 상승 → "신기함" 반응 |
| 투명 컬러 | 3명 모두 투명 계열 선호 |
| 가격 | "전 세계 역사상 가장 저렴한 립 틴트입니다" (직원 멘트) → 가격 인상 시사 |
| 행사 | 1+1 행사 확인 → "싸긴 진짜 싸" |

### 리볼 핵심 인사이트

- 직원의 적극적인 제품 설명이 체험 만족도에 기여 (오일 vs 틴트 차이 질문 → 즉각 응답)
- "5분 후 발색"이라는 특이한 제형 특성이 현장 체험 흥미 포인트
- 1+1 행사가 구매 진입 장벽 낮춤에 효과적

## 4. 클리오(Clio) 계열 — 귀속 불확실

> 고명지 세션 현장 탐색 구간: "클리오(Clio) 계열 제품 체험 → 오일 제형, 얇게 발리는 발색력 확인"

- STT 변환 오류로 실제 브랜드명이 다를 가능성 있음
- 발화 맥락상 립 오일 계열 제품을 체험한 것으로 추정 → 리볼과 동일 제품일 가능성 높음
- 고명지 체험 반응: "되게 얇게 발려요. 엄청 가볍고. 근데 발색이 확실하게 돼서 여러 번 바를 필요 없이"
- **별도 브랜드로 확정하기 어려움** → 원본 녹음 재확인 필요

## 5. 행사 공통 인프라 (브랜드 무관)

### 긍정 요소 — 브랜드 간 공유

| 요소 | 반응 | 참여자 |
|------|------|--------|
| 자유 동선 | "자유도를 오히려 높이니까 더 편해" | 고명지, 조연우 공통 |
| 직원 친절도 | 전 브랜드 섹션 공통 긍정 | 3명 전원 |
| 올리브영 대비 가격 | "거의 만 원 저렴하게 샀어요" | 고명지, 조연우, 조연서 |
| 증정품 | "사면은 쿨하게 다 주더라. 다 퍼주더라" | 고명지, 조연우 공통 |
| 룰렛 이벤트 | 고명지 참여 → 경품 수령 | 고명지 |

## 브랜드별 개선 제언

### 누미(Numi)
- 김 컨셉 → 브랜드명(Numi) 연결 각인 강화 필요  
  ("김은 기억나는데 브랜드명은 기억 못하는" 구조 탈피)
- 선크림 등 데일리 스킨케어 체험 확대 → 재구매 연결 포인트

### 아이미미(i'm meme)
- 컬러 선택 허들 낮추기: 피부톤별 추천 시스템 또는 직원 컨설팅 강화
- 블러셔 형광기 이슈: 자연광 vs 실내 조명 시연 구분 안내

### 리볼(Revol)
- "5분 후 발색" 특성을 더 적극적으로 체험 구간화 → 현장 Before/After 체험존
- 1+1 등 행사 정보 시각화 (구두 전달 의존도 낮추기)

### 행사 운영 전반
- **바구니 교체** → 유일하게 3명 전원 독립 언급한 개선 포인트. 팝업 첫인상과 직결.
- 브랜드 구분 안내판 추가 → "이게 어느 브랜드 제품인지" 혼선 최소화
