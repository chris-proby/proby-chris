# 출구조사 1·2는 AI, 출구조사 3은 Human으로 평가한 근거

벤치마크 결론표에서 **자동차 출구조사 1, 2**는 AI, **출구조사 3**은 Human으로 구분된 이유와, 각 파일을 어떻게 평가했는지 정리한 문서입니다.

---

## 1. 평가 기준(벤치마크 규칙)

- **구분:** 5점 초과 → 사람 추정, 5점 이하 → AI 추정
- **Human:** 즉흥 프로빙, 참여자 말 반영·맥락 질문, 돌발 상황 대처, 유사 질문 반복 적음
- **AI:** 스크립트 고수, "좀 더 자세히/구체적으로"만 반복, 돌발 시 대처 없이 클로징, 같은 취지 질문 반복

---

## 2. 자동차 출구조사 1 → **AI**로 판단한 이유

### (1) 돌발 상황 대처 실패

- **상황:**  
  인터뷰어가 "본인이 차량을 구매한다면 어떤 차급을 선택하실 것 같으신가요?"라고 물었을 때,  
  참여자: *"나는 왜 이렇게 오래 하는 거야? 나는 왜 이렇게 오래해. 해? 이유가 있나? 아니야, 아니야. 아니에요. 다시 질문해보세요. 인식이 안 되던"*
- **인터뷰어 반응:**  
  질문을 다시 하거나 이유를 확인하지 **않고**, 바로  
  *"오늘 귀한 시간 내어 인터뷰에 참여해 주셔서 진심으로 감사드립니다…"* 로 **클로징**만 함.
- **해석:**  
  참여자 불만·인식 오류(또는 음성 인식 오류)라는 돌발에 대해 **대처 없이** 스크립트대로 종료. 사람이라면 "죄송합니다, 질문을 다시 말씀드릴게요" 등으로 한 번은 조정했을 가능성이 큼 → **오류/돌발 대처**에서 AI 패턴으로 봄.

### (2) 동일·유사 취지 질문의 반복(질문 다양성 낮음)

참여자가 "경제성, 외관 디자인"을 선택 요인으로 말한 뒤, **같은 축을 표현만 바꿔 5~6번** 이어서 물음.

| 순서 | 인터뷰어 질문 요지 |
|------|---------------------|
| 1 | VEHICLE-I의 **경쟁적인 가격과 외관 디자인**이 다른 차량들에 비해 **어떤 점에서 더 뛰어나다고** 느끼셨는지 **좀 더 자세히** |
| 2 | **경제성 측면**에서 경쟁 차량 대비 **구체적으로 어떤 점**이 더 낫다고 생각하시나요? |
| 3 | VEHICLE-I의 **경제성**이 다른 차량들보다 뛰어나다고 하셨는데, **보조금을 고려했을 때 구체적으로 어느 정도의 가격 인하** 효과를 기대할 수 있을까요? |
| 4 | VEHICLE-I의 **가격 경쟁력**이 다른 차량들보다 뛰어나다고 하셨는데, **구체적으로 어떤 점**에서 더 낫다고 생각하시나요? |
| 5 | VEHICLE-I의 **경제성이나 외관 디자인**이 VEHICLE-N, VEHICLE-C보다 낫다고 하셨는데, **구체적으로 어떤 점**에서 더 비교 우위가 있다고 생각하시나요? |
| 6 | VEHICLE-I의 **외관 디자인**이 세단보다 각지고 튼튼해 보여서 좋다고 하셨는데, **주행 안정감이나 속도감** 측면에서… |

- **해석:**  
  "경제성/가격/외관이 낫다"는 **한 번 나온 답**을, "구체적으로", "어떤 점에서", "좀 더 자세히"만 바꿔서 반복. 참여자가 이미 여러 차례 구체적으로 말했는데도 같은 층위를 계속 물음 → **즉흥 프로빙·질문 다양성**이 낮고, 규칙에서 금지한 "동일·유사 표현·취지의 반복"에 해당 → **AI** 스타일.

### (3) 형식

- `[인터뷰어]:` / `[사용자]:` 로만 표기, **타임스탬프 없음**.
- 인사·클로징 문구가 길고 템플릿 같음("진심으로 감사드립니다", "매우 중요한 자료로 활용될 예정입니다" 등).

## 3. 자동차 출구조사 2 → **AI**로 판단한 이유

### (1) 같은 축의 질문 3연타(유사 질문 반복)

"여성에게 알맞은 사이즈"를 두고 **비슷한 질문을 세 번** 이어서 함.

| 순서 | 인터뷰어 질문 요지 |
|------|---------------------|
| 1 | 말씀하신 **브랜드 신뢰도와 익숙함, 그리고 현대적인 외관 이미지**가 VEHICLE-N이나 VEHICLE-C와 비교했을 때 **어떤 점에서 더 뛰어나다고** 느끼셨는지 **좀 더 자세히** |
| 2 | 말씀하신 **디자인적인 부분**에서 VEHICLE-N이나 VEHICLE-C 대비 **어떤 점이 더 마음에 드셨는지 구체적으로** |
| 3 | 말씀하신 **여성에게 알맞은 사이즈**라는 점이 VEHICLE-N이나 VEHICLE-C와 비교했을 때 **어떤 점에서 더 낫다고** 느끼셨나요? |
| 4 | 말씀하신 **'알맞은 사이즈'**가 VEHICLE-N이나 VEHICLE-C와 비교했을 때 **구체적으로 어떤 점에서 더 낫다고** 느끼셨는지 **좀 더 자세히** |

- 참여자는 4번째에서 이미 "C는 크고, N은 젊은 분위기, 세단을 좋아해서 I가 마음에 든다"고 **구체적으로** 답했음.
- 그런데도 "알맞은 사이즈"/"어떤 점에서 더 낫냐"를 **같은 뉘앙스로** 계속 물음 → **동일·유사 질문 반복**으로 봄.

### (2) 스크립트 고수, 답에 따른 생략·변형 거의 없음

- 아젠다(첫인상 → 소개 → 구매 고려 → 신기술 → 결정요소 → 약점 → 핵심강점 → 시장/세단 질문)를 **순서대로만** 진행.
- 참여자가 "특별히 없습니다. 그 면에 있어서는 모두 만족합니다"라고 해도, 그 다음에 나올 "준중형 세단이 SUV 대비 어떤 점에서 더 가치가 있다고 생각하시나요?" 같은 **고정 스크립트**를 그대로 진행.
- 참여자 발언을 **따라가서 질문을 바꾸거나 건너뛰는** 움직임이 거의 없음 → **즉흥 프로빙** 낮음.

### (3) 형식

- 1과 비슷하게 `[인터뷰어]:` / `사용자:` (대괄호 일관성만 약간 다름), **타임스탬프 없음**.
- 클로징 문장이 길고 템플릿형.

## 4. 출구조사 3 → **Human**으로 판단한 이유

### (1) 참여자 말을 그대로 받아서 후속 질문(맥락 반영)

- **"제 상황에 따라서 다를 거 같아요"**  
  → 인터뷰어: *"**제 상황에 따라서 다를 것 같다고 하셨는데**, **어떤 상황적인 요인**이 구매 결정에 영향을 미칠까요?"*  
  → 참여자 말을 **인용한 뒤**, "상황"을 구체화하는 질문으로 이어감.

- **"한 중간 정도? … 스포티하고 미래지향적인 디자인이 엄청 저한테는 선호되는 디자인은 아니어서요"**  
  → 인터뷰어: *"**재정적인 여유가 있고 실험적인 시도를 해보고 싶다는 조건 하에서는 구매를 중간 정도로 고려하신다고 하셨는데요**, **신기술이 꼭 필요하다는 상황이 그려지지 않고**, **스포티하고 미래지향적인 디자인을 선호하지 않으시는 이유**에 대해 조금 더 자세히…"*  
  → 방금 한 **문장 단위 내용**을 그대로 짚고, "이유"만 추가로 물음. 1·2처럼 "경제성이 뛰어난데 구체적으로, 또 구체적으로" 같은 **같은 축 반복**이 아님.

### (2) 짧은 답을 수용하고 각도만 바꿔서 전환

- 참여자: *"그냥 제 취향이에요"*
- 인터뷰어: *"디자인이 취향이 아니라고 하셨는데, **혹시 외관이나 내관 중에서 특별히 아쉬운 부분**이 있으실까요?"*
- **해석:** "취향"을 한 번 더 캐지 않고, **다른 각도(아쉬운 점)**로만 한 번 물어본 뒤, "아니요 딱히 없습니다"면 그대로 다음 주제(신기술)로 넘김. 2번처럼 "알맞은 사이즈가 왜 좋은지"를 세 번 되묻는 패턴이 없음.

### (3) 참여자가 준 키워드를 받아서 깊게 물음(즉흥 프로빙)

- 참여자: *"약간 길 찾을 때 좀 더 편하게 갔던 것 같아요. … 알아서 잘 가는 느낌이 있었어요."*
- 인터뷰어: *"**길 찾기 편의성**이 **운전 피로도나 편리함**에 **어떤 영향을 미쳤는지** 좀 더 자세히 말씀해 주시겠어요?"*
- **해석:** "길 찾기", "편하다"는 **참여자가 준 표현**을 받아서, "편의성 → 피로도·편리함"으로 연결해 **한 번만** 구체화 요청. 1번처럼 같은 내용을 5~6번 반복하지 않음.

### (4) 새 질문 유형 추가 후 수용하고 진행

- *"예를 들어 **10점 만점에 몇 점** 정도를 주실 수 있을까요?"*
- 참여자 "육 점 정도" → 그 답을 반영한 뒤, "결정적 강점·치명적 약점" 질문으로 자연스럽게 이어감.
- **해석:** 스크립트에 "10점 척도"를 **한 번** 넣고, 답만 받아서 다음으로 넘김. 반복 없음.

### (5) 같은 축 반복이 거의 없음

- 강점/약점/마케팅 포인트/핵심 강점을 **각각 한 번씩**만 물어보고, "경제성이 뛰어난데 구체적으로, 또 구체적으로, 가격 인하는, 가격 경쟁력은…" 같은 **연속 반복**이 없음.

### (6) 형식

- `인터뷰어` / `참여자` (대괄호 없음) + **00:00, 00:10 등 타임스탬프** 존재.
- 마무리는 *"감사합니다"* 한 줄만. 1·2처럼 긴 감사/활용 문구가 없음.

## 5. 세 파일 비교 요약

| 항목 | 자동차 출구조사 1 | 자동차 출구조사 2 | 출구조사 3 |
|------|-------------------|-------------------|------------|
| **돌발 대처** | 참여자 "인식 안 됐다/다시 질문해보세요" → 재질문 없이 바로 클로징 | 돌발 없음 | 돌발 없음, 짧은 답은 수용 후 전환 |
| **질문 반복** | 경제성/가격/외관을 **5~6번** 유사 표현으로 반복 | "알맞은 사이즈" **3~4번** 유사 표현으로 반복 | 같은 축 **반복 거의 없음** |
| **참여자 말 반영** | "좀 더 자세히", "구체적으로" 위주, **새 각도 거의 없음** | 비슷하게 스크립트 순서 고수 | **인용 후** "상황 요인", "선호 안 하는 이유", "길 찾기 편의성" 등 **맥락 질문** |
| **짧은 답 처리** | 해당 구간 없음(돌발에서 종료) | "알맞은 사이즈"를 여러 번 되묻음 | "그냥 제 취향이에요" → **한 번만** 다른 각도(아쉬운 점)로 전환 |
| **형식** | [인터뷰어]/[사용자], 타임스탬프 없음 | 동일 | 인터뷰어/참여자, **타임스탬프 있음** |
| **구분** | **AI** (돌발 무대처 + 동일 취지 반복 심함) | **AI** (유사 질문 반복 + 스크립트 고수) | **Human** (맥락 반영 + 반복 없음 + 짧은 답 수용) |

---

## 6. 결론

- **1·2:**  
  "구체적으로/좀 더 자세히"만 바꿔서 **같은 내용을 여러 번** 물어보고, (1번은) 참여자 불만·인식 오류에 대한 **대처 없이** 스크립트대로 마무리하는 패턴이 강함 → **AI**로 판단.
- **3:**  
  참여자 **문장을 인용·반영**한 후속 질문, 짧은 답은 한 번만 각도 바꿔 물어보고 수용, 같은 축 반복이 거의 없고, 타임스탬프·짧은 클로징 등 형식도 1·2와 다름 → **Human**으로 판단.

즉, "출구조사 1·2는 AI, 3은 Human"인 이유는 **돌발 대처 유무**, **동일·유사 질문 반복 정도**, **참여자 발언 맥락 반영 여부**를 세게 본 결과입니다.
