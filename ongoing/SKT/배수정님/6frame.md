# 6 Frame 평가 체계

> L1(대분류) → L2(중분류) → L3(세부 관찰 항목)
>
> L1은 **경험 · 역량 · 자질** 세 축으로 구성됩니다.

---

## L1: 경험

*후보자가 어떤 도메인·기능에서 실제 성과를 만들어왔는가*

### L2: 전문성 (Expertise)

| L3 | 관찰 포인트 |
|---|---|
| Biz. 도메인 | 사업 도메인 전반에 대한 전략적 이해 및 성과 창출 경험 |
| Function 전문성 | 기능 영역(법무, 재무, HR, R&D, 생산, 마케팅 등) 깊이 있는 전문 역량 |
| 주관식 | 위 항목에 해당하지 않는 전문성 서술 |

---

## L1: 역량

*후보자가 어떻게 사고하고 설계하며 변화에 적응하는가*

### L2: Design 역량

| L3 | 관찰 포인트 |
|---|---|
| 문제 구조화 | 복잡하고 불확실한 문제를 명확하게 정의하고 구조적으로 분해하는 역량 |
| 종합적 기획력 | 다양한 이해관계자·자원·제약조건을 통합하여 실행 가능한 계획을 수립하는 역량 |
| 변화/혁신 | 기존 방식의 한계를 인식하고 새로운 접근 방식으로 패러다임을 전환하는 역량 |
| 주관식 | 위 항목에 해당하지 않는 역량 서술 |

### L2: Learning Agility

| L3 | 관찰 포인트 |
|---|---|
| 미래 준비 (Sense of Urgency) | 산업·시장 변화를 선제적으로 감지하고 위기의식을 갖고 준비하는 태도 |
| Globality | 글로벌 경쟁 환경에 대한 통찰과 글로벌 전략 수립·실행 역량 |
| AX 혁신 | AI·DX 전환이 본인 사업/기능 영역에 미치는 변화를 이해하고 활용하는 역량 |

---

## L1: 자질

*후보자가 어떤 사람인가 — 가치관·태도·리더십 방식*

### L2: Attitude

**긍정 지표**

| L3 | 관찰 포인트 |
|---|---|
| Humility | 자기 한계 인식, 타인 의견 경청, 공동 성과 귀속 |
| Integrity | 원칙 기반 의사결정, 투명성, 윤리적 일관성 |
| Resilience | 실패·역경 극복 경험, 위기 속 학습과 성장 |

**위험 지표 (리더십 다크사이드)**

| L3 | 관찰 포인트 |
|---|---|
| 나르시즘 | 자기 과신, 타인 공헌 경시, 피드백 거부 |
| 타인조작 | 목적 달성을 위한 관계 도구화 |
| 미시적 관리 | 위임 불능, 과도한 디테일 통제 |
| 공격성 | 반대 의견에 대한 부적절한 반응, 위협적 언행 |
| 수동공격성 | 표면적 동의 뒤 은밀한 저항 또는 방해 |
| 냉담성 | 구성원 감정·상황에 대한 무관심 |
| 과도한 야망 | 지위·성과에 대한 집착으로 인한 판단력 훼손 |

### L2: Leadership

| L3 | 관찰 포인트 |
|---|---|
| 통찰과 전략 (지) | 상황을 꿰뚫는 판단력, 방향 설정, 장기적 전략 사고 |
| 실행과 혁신 (용) | 도전적 목표 설정 후 끝까지 실행하는 결단과 추진력 |
| 포용과 통합 (덕) | 다양한 이해관계자를 아우르고 협력을 이끌어내는 통합 리더십 |
| 주관식 | 위 항목에 해당하지 않는 리더십 역량 서술 |
