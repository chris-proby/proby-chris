# 뷰티팝업 조사 인터뷰 — 조연우 (세션 2)

> **참여자:** 조연우  
> **형식:** AI 인터뷰어 + 현장 실시간 녹음 (Vlog 스타일)  
> **브랜드:** 누미(Numi) 뷰티 팝업 스토어  
> **특이사항:** 화학과 재학생 / 세 명 중 가장 많이 구매 ("나 저기서 큰손이야")

---

## 현장 탐색 (자유 관람 구간)

**주요 관찰 내용:**
- 입장 초반 네이버 라운지 인증 이벤트 진행 → "설명이 좀 있었으면 좋겠는데 이게 뭔지를 모르겠네"
- 브랜드가 여러 개 입점한 구조 파악 → "브랜드가 여러 개가 있는 거구나"
- 리볼(립오일) vs 틴트 차이 직원에게 직접 질문 → 오일 제형 차이 설명 청취
- "전 세계 역사상 가장 저렴한 립 틴트입니다" → 가격 인상 발언
- 1+1 행사 확인 → "싸게, 싸긴 진짜 싸"
- 블러셔 체험: "약간 크림이 있어. 나 요즘 블러셔에 살짝 꽂혀가지고"
- 스틱 쉐딩 탐색 → "턱을 깎고 싶은 나이기 때문에"
- 하이라이터 체험 → "너무 반짝거려. 이건 좀 에바치지"
- 클렌징 오일 → "진심으로 클렌징 오일은 사야 될 것 같은데"
- **바구니 구멍 불편** → "구멍이 너무 커가지고 자꾸 빠진다" (3명 공통 언급)
- 완도 수산시장 컨셉 발견 → "완도 돌김 왜 이렇게 웃기냐"
- 5만 원 이상 구매 → 스쿱 이벤트 참여 (본품을 스쿱으로 직접 담는 이벤트)
- **김 패드 구매** → "집에 패드가 많긴 한데 웃겨서 하나 사봤다, 김"
- 선크림(톤업) 구매 → "아침에 선크림 대신에 출근할 때 그냥 발라볼까 해서"
- 클렌징 밤 → 엄마에게 사줄 목적으로 구매 검토
- "나 10만 원 벌러 왔다가 뭔가 더 쓰고 가는 것 같긴 해" → 임상 아르바이트 동기로 방문했다가 실구매 발생

## Q1. 팝업 스토어 전반 소감

> *Vlog 스타일 촬영*

- 사람이 많아 살짝 부담 → "사람 많은 곳에 너무 안 다녀서 살짝 버거워"
- 동선 자유도: "동선이 정해져 있으면 조금 더 편할 것 같기도 하고, 근데 자유도를 오히려 높이니까 또 그런 건 괜찮기도 하고"
- 참여 이벤트 더 있으면 좋겠다는 의견 → "참여 이벤트 좀 더 있어도 괜찮을 듯"
- 구매 시 증정품이 후함 → "사면은 쿨하게 다 주더라. 올리브영보다 싸고 추가로 더 주니까"
- **김 브랜딩 최고 기억 포인트** → "다른 건 기억 안 나도 김은 기억나. 근데 이 브랜드가 아니라 그냥 김이 기억날 것 같은데"
- 바구니 구멍 불편 재언급 → "처음에 바구니 좀 열받긴 했는데, 그냥 많이 사라고 한 것 같기도 하고"
- 상술을 인식하면서도 많이 구매 → "잔뜩 사간 사람이 되었네요"

## Q2. 가장 인상 깊었던 공간/제품

- **스쿱 이벤트** → 본품을 직접 퍼 담는 참여형 이벤트, 실제로 참여함
- **김 브랜딩 코너** → "나는 김. 귀엽잖아. 김친구가 친절했어. 이제 친구라 해야 될 것 같아"
- 남성 직원 → "김총각. 귀엽다"

## Q4. 팝업 경험 후 브랜드 인식 변화 및 구매 의향

- "앞으로 조금 더 써보고 결정하게 될 것 같긴 한데 일단은 지금 경험은 나쁘지 않은 경험을 했고"
- 젊은 남성 직원 배치 전략 인식 → "확실히 주 고객층이 여성이다 보니까 그런 건 있었고"
- 수산시장 컨셉 → "나름 자기 컨셉에 맞게 해놓은 것 같고"
- 브랜드 설명 보강 필요 → "자기 거에 조금 더 써있으면 모르는 사람들이 많을 수도 있으니까"
- AI 인터뷰 앱 불편 → "이어폰 끼고 있어야 되니까 또 대화해야 되니까 한쪽은 끼고 한쪽은 안 끼고"
- 바구니 구멍 경험 → "솔직한 그 경험은 딱 들어가지는 것 같긴 하고" (솔직한 반응 수집에 긍정적)
