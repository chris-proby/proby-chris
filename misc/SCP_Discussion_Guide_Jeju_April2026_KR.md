# **디스커션 가이드 — 제주 코리도**

리서치 목표:

1. **한국인 여행자가 제주 여행을 위한 펜션을 검색하고 평가하는 방식을 구체적으로 이해하기** – 어떤 필터, 카테고리, 정보 섹션, 소셜 프루프를 활용하는지?

2. **경쟁 플랫폼 대비 Airbnb의 부족한 점 파악하기** \- 제주 펜션 페이지에서 나타나는 Airbnb의 정보 공백 및 UX 마찰 지점

3. **자연스러운 예약 흐름에서 체험 & 서비스 번들 오퍼가 보이는지 진단하기**, 그리고 **제주의 체험 & 서비스 공급량이 쿠폰 사용을 유도할 만큼 충분한지** 평가하기

**섹션 1: 소개 & 워밍업  \[2분\]**

**섹션 2: 여행 계획 & 숙박 우선순위  \[15분\]**

**목표:** 제주 여행의 우선순위와 전반적인 '여행 준비 여정' 이해하기.

* **예정하신 제주 여행에 대해 이야기해 주세요. 어떤 계기로 가시나요? 가장 기대되는 건 무엇인가요?**

  →  누구와 함께 가시나요? 제주는 어떻게 목적지로 정하게 됐나요?

  →  이 여행을 알아보기 시작한 게 얼마나 됐나요?

* **이번 여행이 성공적이었다고 말할 수 있으려면 어떤 조건이 필요한가요?**

  →  꼭 해야 할 것이나 꼭 갖춰야 할 조건이 있나요? 예: 숙소 유형 (펜션 등), 지역, 어메니티 (BBQ 등), 가격, 액티비티 등?

* **제주 여행 계획을 처음부터 이야기해 주세요. 가장 먼저 뭘 알아보시나요? 그다음엔?**

  →  이 여행을 구성하는 요소들과 계획/예약 순서는 어떻게 되나요? → 항공권, 숙박, 액티비티, 렌터카, 레스토랑 예약 등 어떤 순서로?

* **어떤 유형의 숙소에 묵고 싶으신가요?**

  →  펜션에 머물기로 결정하신 이유가 뭔가요? 다른 유형의 숙소에서는 얻을 수 없는, 펜션만의 매력은?

  →  숙소 검색을 위해 어떤 단계를 밟으셨나요? 다음엔 무엇을 하실 건가요? (예: 동행자와 상의, xxx 사이트/앱 방문, 가격 확인, 블로그/리뷰 읽기 등)

  →  제주 여행에서 특별히 중요한 점이 있나요? 다른 목적지에서는 크게 신경 쓰지 않는 것?

* **펜션을 구체적으로 생각하셨을 때, 어떤 펜션에 머물지 선택할 때 가장 중요하게 고려하는 요소는 무엇인가요?**

  →  숙소를 선택할 때 가장 중요한 기준은?

* **위치** – 어떤 위치 조건이 중요하고, 이유는?
* **가격** – 예산은 어느 정도인가요? 적절한 가격을 어떻게 찾으시나요? (예: 적립 포인트, 할인, 여러 플랫폼 가격 비교 등)
* **퀄리티** \- 퀄리티를 판단하는 기준은 무엇인가요? (예: 사진, 리뷰)
* **편안함, 청결도** \- 편안함을 예측하기 위해 어떤 신호를 찾으시나요?
* **어메니티** \- 이번 여행에서 중요한 어메니티는? 왜 중요한가요? 필수 조건인가요?
* **그 외에 다른 것은?**

* **원하시는 펜션의 유형이 따로 있나요?**

  →  감성 펜션은 어떻게 생각하시나요? 어떤 펜션이 '감성 펜션'이라고 생각하시나요? 감성 펜션에 머물고 싶으신가요? 어떻게 검색하시겠어요?

**섹션 3: 경쟁 플랫폼 탐색 (네이버/야놀자/여기어때/아고다)  \[25분\]**

**목표:** 경쟁 플랫폼에서의 검색부터 평가까지 전체 흐름 관찰하기. 질문은 테마 태그로 묶여 있으므로 참여자의 흐름을 따라 먼저 떠오르는 주제로 이동할 수 있습니다.

* **이제, 제주 여행을 위한 \*펜션\*을 어떻게 검색하는지 보여주세요. 평소에 사용하시는 앱이나 웹사이트에서 자유롭게 시작해 주세요. 검색하면서 생각을 소리 내어 말씀해 주세요.**

  →  왜 그 플랫폼에서 시작하셨나요?

  →  \[xxx\]가 항상 이용하는 플랫폼인가요? 가장 먼저 떠오르는 이유가 뭔가요?

**● 홈페이지, 검색 및 리스팅 카드**

* **\[홈페이지\] 다양한 추천, 카테고리, 프로모션이 보이는데요. 평소에 이런 것들을 살펴보시나요?**

  →  얼마나 관련성이 있게 느껴지시나요?

  →  어떤 종류의 캠페인이나 프로모션이 눈에 띄나요? 보통 어떤 걸 찾으시나요?

  →  어떤 '카테고리'나 '추천'이 유용한가요? 클릭해서 들어가시나요, 아니면 그냥 영감을 얻는 용도인가요? 어떤 상황에서 실제로 클릭하시나요?

* **\[검색창\]**

  → **관찰:** 검색어를 직접 입력하는지, 아니면 드롭다운 목적지 목록을 탭하는지. 목록 탭 vs 검색어 입력 중 어떤 걸 선호하나요?

  → **관찰:** 어떤 검색어를 사용하는지. "제주"만 입력하는지, 특정 지역을 입력하는지. 더 구체적인 지역을 검색한다면 왜 그런지 물어보기. 어떻게 그 지역을 알고 검색했나요?


* **\[펜션 카테고리\]**

  → **관찰:** 특정 펜션 카테고리(예: 반려동물 동반, 가족 친화, 풀빌라)를 탭하거나 필터링하는지.

   →  예를 들어 풀빌라, 가족 친화 펜션, 감성 펜션처럼 다양한 '유형'의 펜션을 생각하시나요? **어떻게 정의하시나요? 펜션의 '유형'을 어떻게 구분하시나요?**

   →  플랫폼이 '펜션'이라는 큰 카테고리가 아닌, 이런 펜션 세부 유형까지 '분류'하는 게 얼마나 중요한가요?

  진행자 참고: 많은 경쟁 플랫폼이 펜션 세부 카테고리를 제공합니다. 이것이 한국 여행자의 멘탈 모델에 중요한지 평가하고 있습니다.


* **\[필터\] 어떤 필터를 사용하셨는지, 각 필터가 어떤 역할을 하는지 설명해 주세요.**

  →  필터 없이는 결과를 보지 않을 만큼 절대적인 조건이 되는 필터가 있나요? 그리고 단순히 좁혀나가는 용도의 필터는?

  →  이 필터들이나 표시 방식을 개선할 점이 있나요? 빠진 필터가 있나요?

  →  필터를 건너뛰고 바로 스크롤한다면: 필터 없이 바로 결과를 보셨네요. 평소에도 그렇게 하시나요? 어떻게 좁혀 나가시나요?


* **\[검색 결과\] 이 검색 결과에 대해 어떻게 생각하시나요?**

  →  검색 결과의 리스팅 카드를 보면서 클릭할지 결정하는 데 가장 중요한 정보는 무엇인가요?

  →  체크인 시간 → 이 시점에서 이 정보가 보이는 게 얼마나 중요한가요? 이유는?

  →  리스팅 카드에서 어떤 정보가 더 있었으면 하나요?

**● 리스팅 상세 페이지**

* **이 리스팅에 들어가게 된 이유는? 무엇이 눈에 띄었나요?**

* **이 페이지에서 중요한 정보를 쉽게 찾고 명확하게 확인할 수 있나요?**

  **관찰:** 리스팅 페이지를 스캔하는 순서 기록. 사진으로 바로 스크롤하는지, 어메니티로 넘어가는지, 리뷰를 확인하는지, 가격을 먼저 보는지?

  →  이 페이지에서 어떤 정보가 예약 결정에 자신감을 높이거나 낮게 하나요?


* **\[어메니티\] 이 펜션의 어메니티에 대해 이야기해 주세요. 어떤 어메니티가 중요하고, 그 정보를 찾기 쉬운가요?**

  →  BBQ, 수영장, 자쿠지 등에 대한 정보가 충분한가요? 어떤 정보를 찾고 싶으신가요?

  → 다른 곳에서 확인해야 할 필요가 있나요? (예: 네이버 블로그, 리뷰? 무엇을 위해?)

  → 이 페이지에서 이용 요금을 알 수 있나요? 어디서 찾으시겠어요? 이 정보가 중요한가요?

* **\[페이지 구조\] 이 페이지에서 필요한 정보를 찾는 게 얼마나 쉽거나 어려운가요?**

  →  멀리까지 스크롤해야 찾을 수 있거나, 예상치 못한 곳에 있는 정보가 있었나요? 예: 야놀자는 탭과 구조화된 제목으로 PDP 정보를 구성합니다.


* **\[복수 객실\] 이 펜션은 다양한 객실 타입을 제공하는 것 같네요. 어떻게 비교하고 계신지 이야기해 주세요.**

  →  펜션에 여러 객실 타입이 있는 게 얼마나 흔한가요? 같은 숙소 내 다양한 객실을 비교할 수 있다는 것을 이미 기대하고 계셨나요?

  →  다른 객실 타입을 비교할 때 어떤 요소에 집중하시나요?


* **\[지도\] 지도를 보고 계시네요. 어떤 걸 확인하시나요? 어떤 정보를 얻으려고 하시나요?**

  →  지도에서 정확히 어떤 정보를 찾으시나요? (예: 관광 명소, 해변, 마트 등)

  →  이미 특정 제주 지역을 염두에 두고 계신가요, 아니면 지도가 숙소 위치 결정을 도와주나요?

  →  \[xxx\] 플랫폼 내 지도와 네이버 맵 중 어떤 상황에서 각각을 사용하시나요? 이 플랫폼에서 기대하는 정보와 나중에 네이버에서 확인할 정보는 어떻게 다른가요?


* **\[리뷰\] 리뷰를 어떻게 읽고 계신지 이야기해 주세요. 무엇을 찾고 계신가요?**

  →  어떤 종류의 리뷰 내용이 가장 신뢰감을 주나요? 상세한 텍스트, 사진, 종합 평점, 아니면 다른 것?

  →  사진 리뷰가 평가에 유용하려면 어떤 조건이 필요한가요? (예: 리뷰 수, 사진 카테고리, 사진 품질?)

  →  동영상 리뷰가 있다면: 보시나요? 어떤 동영상 리뷰가 볼 만하고 어떤 건 건너뛰시나요?

* **예약 전에 다른 곳에서 펜션 리뷰를 찾아보시나요? 예: 네이버 블로그, 인스타그램, 유튜브? 이유는?**

  →  외부 SNS로도 리스팅을 검색한다면, 숙박 플랫폼에서 사진/동영상 리뷰를 보여주는 게 얼마나 중요한가요?


* **\[가격\] 이 리스팅의 가격을 어떻게 평가하고 계신지 이야기해 주세요. 어떤 요소를 고려하시나요?**

  →  가격 비교 시 리스팅 페이지에 표시된 가격을 비교하시나요, 아니면 BBQ 비용이나 청소 비용 같은 추가 요금을 포함한 총 비용을 계산하시나요?

  →  이런 추가 요금을 앱에서 예약할 때 바로 지불하는 것과 숙박 중 호스트에게 직접 지불하는 것 중 어떤 방식을 선호하시나요?

* **\[할인/리워드\] 이 페이지의 프로모션이나 리워드에 대해 어떻게 생각하시나요? 결정에 영향을 미치나요?**

  →  네이버 포인트, NOL 포인트 같은 적립 포인트, 리워드, 캐시백은 어떻게 계산에 반영하시나요? 최종 '가치'를 어떻게 계산하시나요?

  →  쿠폰을 언급한다면: 예약 전에 적극적으로 쿠폰을 찾으시나요, 아니면 플랫폼이 보여주는 쿠폰에 의존하시나요? 쿠폰을 찾는 흐름은 어떻게 되나요?


**섹션 4: Airbnb 탐색  \[20분\]**

**목표:** Airbnb에서 동일한 검색 과제 수행. 방금 경험한 경쟁 플랫폼과 비교해 마찰 지점과 정보 공백 관찰하기.

* **Airbnb를 사용해 보셨거나 들어본 적 있으신가요?**

  →  Airbnb 하면 무엇이 떠오르시나요?

  →  이번 제주 여행에서 Airbnb를 고려하셨나요? 이유는?


* **이제 Airbnb를 열고, \[xxx\]에서 하셨던 것처럼 제주 여행을 위한 펜션을 검색해 주세요. 첫인상을 소리 내어 말씀해 주세요.**


**● 홈페이지, 검색 및 리스팅 카드**

* **\[홈페이지\] Airbnb 홈페이지를 보면 무엇이 눈에 띄나요? \[xxx\]와 비교하면 어떤가요?**

  →  관련성 있게 느껴지는 카테고리나 추천이 있나요? 이유는?

  →  홈페이지만 보면 Airbnb에서 원하는 펜션을 찾을 수 있을 것 같은가요?


* **\[검색창\] Airbnb 홈페이지를 보면 무엇이 눈에 띄나요? \[xxx\]와 비교하면 어떤가요?**

  → 검색창 경험을 개선하고 싶은 부분이 있나요?


* **\[필터\] 사용하시는 필터에 대해 이야기해 주세요. 필요한 필터가 있나요?**

  → **관찰:** 어떤 유형의 필터를 사용하는지? 펜션 유형 필터가 없다면 어떻게 대처하는지? 대안적인 검색 전략은?

  → \[xxx\]에서 \[특정 필터/카테고리\]를 사용하셨는데, 여기도 동일한 기능이 있나요? 없다면 어떻게 대처하시나요?

  → Airbnb에서 \[xxx\]에서 봤던 것과 같은 수준의 펜션 세부 카테고리가 있는 게 얼마나 중요한가요? (예: 가족 친화, 반려동물 동반 등)

  → 더 있었으면 하는 필터가 있나요? 그게 왜 중요한가요?


* **\[검색 결과/리스팅 카드\] \[xxx\]와 비교해 이 검색 결과는 어떻게 생각하시나요?**

  → Airbnb 리스팅 카드의 사진 품질, 가격 표시, 정보에서 눈에 띄는 차이점이 있나요?

  → 리스팅 카드에서 중요한 정보가 빠져 있는 것은? (예: Airbnb P2에는 체크인 시간이 표시되지 않습니다. 이게 중요한가요? 그 외에는?)

* **이 리스팅 페이지를 탐색해 주세요. \[경쟁 플랫폼\]과 비교해 무엇이 눈에 띄나요?**

  **관찰:** 처음 무엇을 보는지. 스캔 패턴이 \[경쟁 플랫폼\]에서와 같은가요? 레이아웃이 행동에 영향을 미치나요?

  →  \[경쟁 플랫폼\]에서는 쉽게 찾을 수 있었는데 여기서 찾기 어려운 정보는? (또는 그 반대는?)


* **\[어메니티\] 어떤 어메니티 정보를 찾으려 하시나요? Airbnb에서 쉽게 찾을 수 있나요?**

  →  어느 정도의 상세함이 중요한가요? (예: BBQ라면 비용? 사용 방법? 어떤 정보가 꼭 필요하고 있으면 좋은 수준인가요?)

  →  페이지 상단에 적절한 어메니티가 강조되어 있나요? 바꾸고 싶은 것이 있다면?

* **\[추가 비용\] 이런 어메니티의 이용 요금이 Airbnb에서 명확하게 표시되어 있나요?**

  →  \[xxx\]에서 이 정보를 표시하는 방식과 비교해 어떤가요? 이 정보를 표시하는 가장 좋은 방법은?

  → **관찰:** 많은 경우 호스트가 비구조화된 텍스트 섹션에 비용을 기재합니다. 참여자가 이것을 문제로 느끼는지 관찰하기.

  →  비용이 명확하지 않다면 어떻게 하시겠어요? (예: 다른 리스팅 찾기, 호스트에게 연락, 다른 플랫폼 확인?)


* **\[페이지 구조\] 이 페이지에서 정보를 찾는 게 얼마나 쉽거나 어려운가요?**

  →  페이지를 재구성할 수 있다면 무엇을 옮기거나 바꾸시겠어요?


* **\[복수 객실\] 이 펜션에 다양한 객실 타입이 있는지 알 수 있나요?**

  → **관찰:** 현재 Airbnb는 각 객실을 별도 리스팅으로 표시합니다. 같은 펜션 내 다른 객실을 확인하는 쉬운 방법이 없습니다 (호스트 프로필을 통해 다른 리스팅을 확인하는 방법만 있음).

  →  \[경쟁 플랫폼\]에서 펜션을 검색할 때 같은 숙소 내 다양한 객실 타입을 비교하셨나요? 여기서도 그런 생각을 하셨나요?


* **\[지도\] Airbnb에서 지도 경험은 어떤가요? 어떤 정보를 얻고 계신가요?**

  **관찰:** Airbnb와 경쟁 플랫폼에서 지도를 다르게 활용하는지?

  →  펜션의 위치 맥락을 파악하기가 얼마나 쉽거나 어려운가요? 어떤 어려움이 있나요?

  →  여기 지도는 구글 맵을 사용합니다. 이 지도에서 위치 맥락이 충분한가요? 네이버 맵에서 따로 확인해야 할 것은 무엇인가요?


* **\[리뷰\] Airbnb 리뷰는 \[xxx\]와 비교해 어떤가요? 눈에 띄는 점을 이야기해 주세요.**

  **관찰:** 여기서 리뷰에 더 많은 시간을 쓰는지, 적게 쓰는지? 사진 리뷰를 특별히 찾는지?

  →  이 리스팅의 경우, 리뷰만으로 퀄리티를 잘 파악할 수 있나요? 부족하다면 무엇이 빠져 있나요? 더 많은 리뷰를 찾으러 어디로 가시겠어요?

  →  Airbnb 리뷰가 더 유용해지려면 어떻게 해야 할까요?

* 누구의 리뷰? (예: 한국인 리뷰만, 커플 여행자 리뷰)
* 형식 (예: 사진, 동영상)
* 분류 (예: 테마별로 구조화?)
* 내용 (예: 더 긴 리뷰?)
* 수량 (자신감을 느끼려면 몇 개의 리뷰가 필요한가요?)

  →  Airbnb가 사진/동영상 리뷰에 더 투자하고자 한다면, 한국 여행자에게 더 유용하게 만들기 위한 조언을 해주신다면?

* **\[가격\] 가격을 보면, \[xxx\]에서 본 비슷한 펜션과 비교해 어떤가요?**

  →  어떤 차이점이 눈에 띄나요? (예: 기본 인원 가격 / 총 인원 가격)

  →  Airbnb에 표시된 가격에 대해 궁금한 점이 있나요? 불분명한 것은?

  →  Airbnb가 \[xxx\]에 비해 비슷한 가치를 제공한다고 느끼시나요? Airbnb가 대안보다 더 좋은 가치처럼 느껴지려면 무엇을 제공해야 할까요?


* **두 플랫폼 모두 검색해 보신 후, 차이점을 요약하신다면?**

  →  Airbnb가 제주 펜션 예약에서 선택받으려면 가장 먼저 개선해야 할 한 가지는?

  →  Airbnb가 야놀자보다 더 잘하는 점이 눈에 띄었나요?
