# short-drama-guide

Static English participant guide for the short-drama / Vigloo study (Proby interview flow, Vigloo install, rating scale).

## Live site

**Production:** [https://short-drama-guide.vercel.app](https://short-drama-guide.vercel.app)

## Deploy

From this directory (team scope is required for this account in non-interactive mode):

```bash
npx vercel deploy --prod --yes --scope churryboys-projects-9bab66ff
```

Or after `npx vercel link` once:

```bash
npx vercel --prod
```

## Update content

Edit `index.html` in this folder (or sync from `ongoing/스푼랩스/흥미도 트래커/참가자_시청_안내_가이드_en.html`) and redeploy. Image assets live alongside `index.html`.
